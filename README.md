# VISION  
### Virtual Infrastructure Surveillance & Intelligence Optimization Network

VISION is a **Smart City Infrastructure Intelligence Platform** designed to help city authorities monitor, manage, and analyze urban infrastructure in real time.

The platform provides a centralized dashboard for tracking infrastructure assets such as roads, streetlights, pipelines, and public utilities while visualizing their health and operational status.

---

## 🚀 Project Overview

Urban infrastructure management is often reactive and fragmented.  
VISION transforms this process by providing a **data-driven, centralized monitoring system** for city infrastructure.

The system enables authorities to:

- Monitor infrastructure assets in real time
- Track operational status and health score
- Visualize assets geographically on a city map
- Manage infrastructure lifecycle efficiently
- Improve maintenance planning

---

## ✨ Key Features

### 📊 Infrastructure Dashboard
A centralized dashboard displaying:

- Total infrastructure assets
- Operational infrastructure
- Assets under maintenance
- Critical infrastructure
- Reported issues

---

### 🏗 Infrastructure Asset Management

Authorities can:

- Register new infrastructure assets
- View infrastructure inventory
- Update asset status and health score
- Track infrastructure location

Supported asset types include:

- Roads
- Streetlights
- Bridges
- Pipelines
- Public facilities

---

### 🗺 Smart City Map Visualization

VISION includes an **interactive infrastructure map** that:

- Displays all registered assets
- Uses map markers to represent infrastructure
- Shows asset health and operational status
- Enables geographic infrastructure monitoring

Powered by **Leaflet.js and OpenStreetMap**.

---

### ⚙ Infrastructure Health Monitoring

Each asset has a **health score** indicating its condition.

Health indicators:

- 🟢 Healthy
- 🟡 Warning
- 🔴 Critical

This helps prioritize infrastructure maintenance.

---

### 🌙 Modern Smart City Dashboard UI

VISION includes a modern admin dashboard with:

- Dark / Light mode toggle
- Responsive UI
- Interactive cards
- Modern asset management interface
- Clean navigation system

---

## 🧱 System Architecture

Admin opens VISION Dashboard

Admin registers infrastructure assets ↓

Asset data stored in Firebase Firestore ↓

Dashboard updates infrastructure statistics ↓

Infrastructure assets appear in Asset Table ↓

Assets are visualized on Smart City Map ↓

Health score indicates infrastructure condition
