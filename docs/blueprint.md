# **App Name**: GestureMate

## Core Features:

- Gesture Recognition: Hand gesture recognition via webcam feed, using a machine learning model to interpret gestures.
- Action Mapping: Map recognized hand gestures to predefined actions such as moving the cursor, clicking, scrolling, or other specific commands.
- Visual Feedback: Display a live video feed of the webcam with an overlay that indicates the recognized gesture.
- Calibration Settings: Calibrate and adjust gesture sensitivity settings.
- Helpful assistant: AI tool powered assistant can help the user decide when is the best time to use the app

## Style Guidelines:

- Primary color: Blue (#4681C0), providing a sense of calm and reliability, reflecting the stable technology.
- Background color: Light gray (#F0F4F8), almost white but a little more forgiving.
- Accent color: Orange (#C06546), drawing attention to controls or gesture prompts without disrupting.
- Font: 'Inter' (sans-serif) for clear, readable text in UI elements and instructions.
- Simple, easily recognizable icons to represent actions (e.g., cursor, click, scroll) in the settings or help sections.
- Clean, uncluttered layout with video feed prominently displayed. Place the controls and settings unobtrusively to avoid distracting the user.
- Subtle animations or transitions to indicate successful gesture recognition or when executing mapped actions.
